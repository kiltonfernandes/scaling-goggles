# YouTube → Podcast Feed

Auto-generated by [Lovable](https://lovable.dev).

This repo turns YouTube channels into a real podcast RSS feed that works with
**Snipd, Apple Podcasts, Pocket Casts, Overcast**, etc.

## How it works

- `channels.json` — list of channels (managed by the Lovable UI)
- `.github/workflows/sync.yml` — runs every 12h; downloads new videos as MP3 and updates the feed
- `.github/workflows/purge.yml` — runs daily; deletes MP3s older than 48h to save space
- `docs/feed.xml` — the podcast RSS feed, served by GitHub Pages

## Your feed URL

```
https://<your-user>.github.io/<repo-name>/feed.xml
```

Paste it into Snipd / Apple Podcasts / Pocket Casts → "Add by URL".

## Audio quality

96 kbps mono MP3 (optimized for spoken-word podcasts, ~40 MB/hour).
Editable in `channels.json` under `settings.audio_bitrate_kbps`.
